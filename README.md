# 공부했뇌

## 프로젝트 소개

공부한 내용을 기록하고 학습 시간을 관리할 수 있는 웹 서비스입니다.  
공부 시간에 따라 경험치가 증가하며, 뇌 캐릭터가 성장하는 기능을 통해 학습 동기를 부여합니다.

## 주요 기능

- 공부 기록 추가
- 학습 시간 측정
- 경험치 및 레벨 시스템
- 뇌 캐릭터 성장
- 최근 7일 학습 통계
- 과목별 학습 비율 확인
- 복습 기능

## 사용 기술

- React
- JavaScript
- CSS
- LocalStorage
